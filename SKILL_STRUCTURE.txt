🎯 DEPLOYMENT-ABLAUF (Zusammenfassung)

1. Kamera-Info sammeln (IP, RTSP-URL, Credentials)
2. (Optional) Telegram Bot einrichten
3. .env.production mit Werten ausfüllen
4. python test_camera.py → MUSS PASS sein
5. docker-compose up -d → System starten
6. docker logs -f stallwache → Logs überwachen
7. bash health_check.sh → Verifikation
8. Fertig! System läuft 24/7
