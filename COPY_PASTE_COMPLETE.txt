================================================================================
TROUBLESHOOTING
================================================================================

If you got an ERROR anywhere:

ERROR: "fatal: not a git repository"
→ Make sure you ran "git init" first
→ Make sure you're in the Stallwache directory
→ Try: pwd (should end with "Stallwache")

ERROR: "git: command not found"
→ Git is not installed
→ Download from: https://git-scm.com/download
→ Install it
→ Restart Terminal
→ Try again

ERROR: "failed to push"
→ Make sure YOU REPLACED YOUR_USERNAME
→ Make sure the GitHub repo exists
→ Try: git remote -v (check it shows your repo)

ERROR: "Permission denied"
→ GitHub will ask for your credentials
→ Enter your username and password/token
→ Or use GitHub's password manager

ERROR: "fatal: 'origin' does not appear to be a 'git' repository"
→ You forgot Section E (git remote add origin)
→ Go back and do that section
